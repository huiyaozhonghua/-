# 最强天团新达人导航 项目结构

## 项目简介
本项目旨在为用户提供一个全面的达人导航平台，聚合各类达人资源。

## 目录结构
```
- src/            # 源代码目录
    - components/ # 组件文件
    - assets/     # 资源文件
    - pages/      # 页面的实现
- public/         # 公共文件
- README.md       # 项目说明文件
```

## 技术栈
- **前端**: React, Vue
- **后端**: Node.js, Express
- **数据库**: MongoDB

## 安装步骤
1. 克隆项目
2. 运行 `npm install` 安装依赖
3. 启动服务 `npm start`
